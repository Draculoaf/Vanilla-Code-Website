For this website, I wanted to create an aesthetic which was clean, and minimalist. To do this, I chose to use very few colours throughout my website, keeping it to cool shade or black and white. The navigation bar is one the left side of the page both in computer size and in responsive mode - this is to help set up expectations for the user. I added a footer at the end of every page to make sure that the user knows when the content is done. Furhtermore, I have added both underlining on the active page, and lightened the text when uses hover their cursor over the navigation elements to assist with expectations. The website has no particular font, and this is because I have used font sizing to ensure that the user can distinguish between the headings and the content.
